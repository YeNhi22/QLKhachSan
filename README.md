# 🏨 Hệ thống quản lý khách sạn

## 📌 Giới thiệu dự án

Hệ thống **Quản lý khách sạn** được xây dựng nhằm tự động hóa các nghiệp vụ quản lý khách sạn, bao gồm: quản lý đặt phòng, check-in/check-out, quản lý dịch vụ, thanh toán hóa đơn, và báo cáo thống kê. ## 🎯 Mục tiêu

- Xây dựng hệ thống quản lý khách sạn hiện đại, tự động hóa và thân thiện với người dùng.
- Hỗ trợ các nghiệp vụ chính: đặt phòng, check-in/out, dịch vụ, hóa đơn, báo cáo.
- Đảm bảo tính chính xác, nhanh chóng và khả năng mở rộng trong tương lai.

## 🔧 Chức năng chính

### 👤 Khách hàng
- Đăng ký / Đăng nhập tài khoản
- Xem danh sách và chi tiết phòng
- Đặt phòng / Hủy đặt phòng
- Yêu cầu dịch vụ / nhân viên buồng phòng
- Gửi phản hồi, đánh giá
- Thanh toán hóa đơn

### 🧑‍💼 Nhân viên lễ tân
- Xử lý đặt phòng
- Check-in / Check-out khách hàng
- Quản lý hóa đơn
- Hỗ trợ tư vấn khách hàng

### 📊 Quản lý khách sạn
- Quản lý tài khoản nhân viên
- Quản lý phòng, nhà cung cấp
- Theo dõi tình trạng phòng
- Xem báo cáo thống kê (doanh thu, công suất phòng...)
- Phân quyền hệ thống

### 🧹 Nhân viên buồng phòng
- Dọn dẹp phòng
- Cập nhật trạng thái phòng
- Báo cáo vật dụng hỏng
